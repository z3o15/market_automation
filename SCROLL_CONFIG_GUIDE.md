# 滑动配置说明

## 当前滑动距离

项目中有两种滑动距离配置：

### 1. 200像素滑动（可配置）
- 滑动起始位置：y = 900（屏幕中下部）
- 滑动结束位置：y = 700（屏幕中上部）
- 滑动距离：900 - 700 = 200像素
- 配置位置：[`config/market_config.json`](config/market_config.json:30-37) 中的 `scroll_start` 和 `scroll_end`
- 这个距离适合小幅滚动，可以精确控制页面内容的位置

### 2. 800像素滑动（固定）
- 滑动起始位置：y = 900（屏幕中下部）
- 滑动结束位置：y = 100（屏幕上部）
- 滑动距离：900 - 100 = 800像素
- 配置位置：固定在 [`market_automation/market_clicker.py`](market_automation/market_clicker.py:191-227) 的 `scroll_up_800_pixels()` 方法中
- 这个距离适合大幅滚动，可以快速翻页

## 修改滑动距离的方法

### 方法一：修改配置文件（推荐）

编辑 `config/market_config.json` 文件中的以下配置：

```json
"scroll_start": {
  "x": 358,
  "y": 894  // 修改此值可调整滑动起始点
},
"scroll_end": {
  "x": 358,
  "y": 20   // 修改此值可调整滑动结束点，从而改变滑动距离
}
```

**修改建议：**
- 增加滑动距离：减小 `scroll_end.y` 的值或增大 `scroll_start.y` 的值
- 减少滑动距离：增大 `scroll_end.y` 的值或减小 `scroll_start.y` 的值
- 保持水平位置不变：两个点的 `x` 坐标应保持一致

### 方法二：修改代码中的默认值

编辑 `market_automation/market_clicker.py` 文件第41-42行：

```python
'scroll_start': self.config.get('scroll_start', {'x': 358, 'y': 894}),  # 修改这里的默认y值
'scroll_end': self.config.get('scroll_end', {'x': 358, 'y': 20}),      # 修改这里的默认y值
```

## 滑动持续时间

当前滑动持续时间为 **500毫秒**，在 `market_automation/market_clicker.py` 第137行：

```python
duration=500  # 滑动持续时间500毫秒，可根据需要调整（值越大滑动越慢）
```

**修改建议：**
- 增加持续时间：滑动速度变慢，适合精确控制
- 减少持续时间：滑动速度变快，适合快速滚动

## 常见滑动距离参考

根据不同需求，可以设置以下滑动距离：

| 用途 | 起始y | 结束y | 滑动距离 | 说明 |
|------|-------|-------|----------|------|
| 微小滑动 | 900 | 800 | 100像素 | 适合精细调整 |
| 小幅滑动 | 900 | 700 | 200像素 | 当前可配置设置，适合精确控制 |
| 中等滑动 | 900 | 600 | 300像素 | 适合微调位置 |
| 大幅滑动 | 900 | 400 | 500像素 | 适合翻半页 |
| 超大滑动 | 900 | 100 | 800像素 | 当前固定设置，适合翻整页 |
| 最大滑动 | 900 | 0 | 900像素 | 最大滑动距离 |

## 注意事项

1. 坐标系统：屏幕左上角为原点(0,0)，向右为x轴正方向，向下为y轴正方向
2. 屏幕尺寸：不同设备的屏幕尺寸可能不同，需要根据实际情况调整坐标
3. 滑动范围：确保坐标值在屏幕范围内，避免滑动到屏幕外
4. 测试验证：修改后建议先测试，确保滑动效果符合预期
